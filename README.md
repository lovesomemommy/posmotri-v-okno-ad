# H1 posmotri-v-okno-ad

**Описание проекта**

«Посмотри в окно» — это учебный проект от Яндекс Практикума, в котором реализована стилизованная веб-страница с видеоплеером и списком карточек. Проект тренирует работу с HTML и CSS, умение точно воспроизводить макет, а также применять актуальные возможности CSS:

flexbox

grid

псевдоклассы (:focus-visible, :has, :active, :hover)

работа с кастомными чекбоксами

кастомизация скроллбара

использование современных единиц (inline-size, block-size)

выстраивание адаптивных блоков без изменения HTML-структуры

Проект решает ключевую задачу: научиться верстать интерфейсы, точно соответствующие макету, используя современные возможности CSS и избегая лишних обёрток в HTML.

В отличие от предыдущих учебных работ, здесь упор делается на:

использование сложных состояний элементов интерфейса

создание полностью кастомного интерактивного чекбокса

работу со сложной сеткой, где нельзя менять HTML

необходимость добиться точного совпадения размеров и выравнивания по макету

грамотную работу с focus и focus-visible, обеспечивая доступность интерфейса

**Использованные технологии**

HTML

CSS

Flexbox

CSS Grid

Псевдоклассы: :hover, :focus, :focus-visible, :active, :has

Кастомные скроллбары

object-fit, object-position для работы с видео

inline-size / block-size

Семейства шрифтов:

Fira Sans Condensed

Oswald

**Инструкция по использованию**

Введите текст в поле поиска — срабатывает кастомное оформление input.

Отметьте чекбоксы — отображается кастомная заливка.

Наведите на текст или карточки — появляются эффекты hover.

Навигация клавиатурой работает корректно благодаря использованию :focus-visible.

Карточки появляются справа, видео — слева, соответствуя макету.

При переполнении списка появляется кастомный вертикальный скролл

**Как запустить проект**
Требования

Браузер с поддержкой современных CSS-фич:

:has()

focus-visible

свойств inline-size, block-size

Желательно Chrome ≥ 105, Firefox ≥ 121 или любой современный Chromium-браузер.

Проект не требует сборщиков, библиотек или серверной части.
